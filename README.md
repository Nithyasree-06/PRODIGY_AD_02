FlutterFlow To-Do App (UI Prototype)

This project is a simple **To-Do List mobile app UI** designed using **FlutterFlow**.

Features
- Home screen with task list
- Add task screen
- Edit task screen (UI only)
- Delete task option (UI level)
- Clean and simple layout

Tools Used
- FlutterFlow (No-code / Low-code)

Project Type
- UI Prototype
- No backend integration
- No Firebase
- No APK export (FlutterFlow free plan)

Learning Outcomes
- FlutterFlow UI builder
- Page navigation
- State handling (local)
- Designing mobile app layouts
- Understanding app flow
